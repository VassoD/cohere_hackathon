# cohere_hackathon

The web app is running with Flask

You need python installed in your pc (https://www.python.org/downloads/)

You need pip:
python get-pip.py

To install Flask:
pip install Flask

git clone the repo
cd in the repo (cd cohere_hackathon)

to run the app:
FLASK_APP=main.py FLASK_ENV=development flask run
or python3 main.py
